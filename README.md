# node-matrix-assets

Requireable object list of Squiz Matrix core assets.

Extracted using the [JS API getAssetTypes] function.

# Usage

```js
var assets = require('node-matrix-assets');

console.log(assets['page_standard'].name);

// 'Standard Page'
```
